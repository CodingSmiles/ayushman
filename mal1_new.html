MAL1 - MAIN AVIONICS LOGICBOARD 1

The MAIN AVIONICS LOGICBOARD 1 is the main avionics computer of the rocket AGM1. MAL1 completes the purpose of AGM1/Autonomously Guided Module which is to record essential data. MAL1 records flight data like, Alitude, Temprature and Atmospheric Pressure using a BMP280 and logs it into a SD card using a SD card adapter. MAL1 uses an arduino nano to complete all of its on-flight tasks.

The software on MAL1 takes care of the Data being recorded. The Flight Software tracks data while the rocket is powered on. The software is coded using a variant of C++ using the Arduino IDE software.

SOFTWARE/CODE

DATA

MAL1 records data using just one small component, the BMP280. The BMP280 recieves all essential data and sends it to the arduino which later sends it to the sd card adapter. This data is later saved in a .DATA file to be later extracted into the computer.
